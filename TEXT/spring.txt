+ Spring STS tool
 - STS : Spring Tool Suite
 
 - 이클립스에 Spring Framework 를 개발할 수 있도록 제공되는 툴
 
+ STS 배포 방식 2가지

 1. 이클립스에 플러그인 형태로 STS 툴을 설치하는 방법
 
 2. STS tool 을 다운받아 한번에 설치하는 방법
  ( 이클립스 + STS 를 한 묶음으로 만들어 스프링 전용 툴을 만든 것 )
